This is the KiwiGrid profiling part of the JouleUnit framework.

KiwiGrid is a company providing simple power meters that can be used
to profile electronic devices via the plug to their continuous power
supply.

This profiling part requires an SNMP library that can be optained
from http://www.snmp4j.org/

JouleUnit is an extension of the popular JUnit framework that allows 
testing Java applications w.r.t. their energy consumption.

JouleUnit is published under GPL. More details on the GPL can be found
in the LICENSE.txt
