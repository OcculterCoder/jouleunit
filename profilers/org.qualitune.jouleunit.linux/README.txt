This is the Linux profiling part of the JouleUnit framework.

Requires a running Linux on the device the test cases are executed on
and a running ACPI service (battery sensor in power management).

JouleUnit is an extension of the popular JUnit framework that allows 
testing Java applications w.r.t. their energy consumption.

JouleUnit is published under GPL. More details on the GPL can be found
in the LICENSE.txt
